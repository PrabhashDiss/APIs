# APIs
This document explains the API for accessing the data.

# API Endpoints

## GET /api

### Description
Returns a "Hello, World!" message.

### Request
None

### Response
200 OK
Hello, World!

## GET /api/data

### Description
Returns a JSON object with data.

### Request
None

### Response
200 OK
{
  "foo": "bar",
  "baz": 42
}
